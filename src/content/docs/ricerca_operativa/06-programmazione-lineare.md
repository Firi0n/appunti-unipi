# Invuluppo convesso

Il più piccolo insieme di punti racchiuso dai vertici.

# Vertice

Punto che non può essere riscritto come combinazione convessa di due punti diversi da se stesso.

# Poliedro

Politopo + Cono.

# Politopo

Risultato dell'invuluppo convesso.

# Inviluppo conico

è un inviluppo convesso senza il limite che $\sum\lambda= 1$.

# Cono

Risultato dell'inviluppo conico.
