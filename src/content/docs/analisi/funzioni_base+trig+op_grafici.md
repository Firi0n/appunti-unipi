# Funzioni elementari

## Funzioni lineari

$$y=ax+b$$

-   $a = \frac{\Delta y}{\Delta x}=\text{coefficiente angolare}$
-   b = termine noto.

![Funzioni lineari](/src/assets/analisi/funzioni_lineari.png)

### Funzine inversa

$$y=x^\frac{1}{k}$$

k = pari $\Rightarrow$ la funzione è invertibile nel dominio $[0,+\infty)$.

# Funzioni trigonometriche

## Seno

$$\sin(x):\R\to[-1,1]$$

periodica di periodo $2\pi$.

è **surgettiva** e se si restringe il dominio a $[-{\pi\over 2}, {\pi\over 2}]$ diventa:

-   **Iniettiva**;
-   **Strettamente crescente**.

![img](/src/assets/analisi/seno.png)

## Arcoseno

$$\arcsin(x):[-1,1]\to[-{\pi\over 2}, {\pi\over 2}]$$

Inversa del seno.

![img](/src/assets/analisi/arcoseno.png)

## Coseno

$$\cos(x):\R\to[-1,1]$$

periodica di periodo $2\pi$.

è **surgettiva** e se si restringe il dominio a $[0, \pi]$ diventa:

-   **Iniettiva**;
-   **Strettamente crescente**.

![img](/src/assets/analisi/coseno.png)

## Arcocoseno

$$\arccos(x):[-1,1]\to[0, \pi]$$

Inversa del coseno.

![img](/src/assets/analisi/arcocoseno.png)

## Tangente

$$\tan(x):\R-\{{\pi\over 2}+k\pi\}\rightarrow\R$$ $$\tan(x) = {\sin(x)\over\cos(x)}$$

periodica di periodo $\pi$.

è **surgettiva** e se si restringe il dominio a $(-{\pi\over 2}, {\pi\over 2})$ diventa:

-   **Iniettiva**;
-   **Strettamente crescente**.

![img](/src/assets/analisi/tangente.png)

## Arcotangente

$$\arctan(x):\R\to(-{\pi\over 2}, {\pi\over 2})$$

Inversa della tangente.

![img](/src/assets/analisi/arcotangente.png)
